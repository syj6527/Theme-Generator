# 🌫️ 테마 생성기 (Theme Generator) — SillyTavern Extension

키워드/장면을 입력하면 그 **공기**의 색을 뽑아 ST 채팅 테마(12색 + 폰트)를 만들고 네이티브 색상 피커에 주입한다.

## 철학 — 색 생성기가 아니라 무드(공기) 생성기
"청포도"를 넣으면 초록을 뽑는 게 아니다. **"청포도가 어디 있는 순간인가"**를 먼저 상상한다.
- 새벽 냉장고 속 청포도 → 냉장고 블루 / 얼음 화이트 / 젖은 그레이, 초록은 형광펜에만
- 한여름 청포도밭 → 햇빛 크림 / 마른 흙 / 쨍한 연두
같은 키워드도 장면 따라 완전히 다른 테마. 키워드는 주연색이 아니라 포인트일 뿐, 장면의 공기가 배경·본문을 지배한다.

## 두 탭
### 🌫️ Scene (메인)
```
키워드 "청포도 여름 얼음"
  → AI가 '장면'을 그림 (색 언급 안 함)  ← 직접 써도 됨 (예: 비 온 뒤 흐린 하늘)
  → 장면을 7축으로 분류 (온도·자연광·인공조명·습도·공기·채도·무드)
  → 엔진(결정론적)이 그 축으로 색 + 색이름 생성
  → 명도순 12슬롯 배치 + 형광펜 4타입 + 폰트
```
- **AI는 장면과 축만, 색은 100% 엔진** → hex 수렴 없음, 무한 조합
- "먹구름"은 태그가 아니라 차가움+습함+흐림+저채도의 **조합**으로 분해
- 다이스 3단: 🎲새 장면(전부 새로) / 🎲색상(축 고정, 색만) / 🎲분배(색 고정, 배치만)
- 고급: 밝기·채도·몽환 미세조정 슬라이더
- 색마다 이름(얼음 화이트, 젖은 콘크리트, 새벽 라벤더…) — 무드 라이브러리 자산
- AI 미연결 시 소분류 프리셋(비온뒤/새벽냉장고/도서관…)으로 폴백

### 🌧️ Mood
무드 축 6개 슬라이더 + 디저트 프리셋 12종. 결정론적.

## 공유
두 탭은 같은 12슬롯·미리보기·ST 적용(피커 주입)·라이브러리 공유.
네이티브 10슬롯은 색상 피커에, 커스텀 2슬롯(대사색/형광펜)+폰트는 주입 style로.

## 설치
`SillyTavern/data/<user>/extensions/theme-gen/`에 폴더째 → ST 재시작.
