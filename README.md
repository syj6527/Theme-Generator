# 🌧️ 테마 생성기 (Theme Generator) — SillyTavern Extension

무드/소재를 입력하면 ST 채팅 테마(12색 슬롯 + 폰트)를 통째로 만들어 네이티브 색상 피커에 주입한다.

## 두 가지 모드 (탭)

### 🌧️ Mood
무드 축 6개(온도·습도·밝기·채도·대비·환상)를 슬라이더로 조절 → 결정론적 엔진이 12슬롯 계산.
디저트 프리셋 12종, 폰트 4종, "인용=블루" 토글. AI 연결 시 장면 텍스트 → 축 자동 추출(옵션).

### 🍵 Object
소재/색을 띄어쓰기로 입력(예: `먹구름 노을 서리`). 핵심 원리:
- **소재 = 단일색이 아니라 색상 범위.** 먹구름도 여름폭우/겨울새벽/바다/노을먹구름이 다 다른 색 → 매번 범위에서 랜덤 추출.
- **밴드 하모나이저** — 채도는 좁은 밴드로 묶어 톤 통일, 명도는 순위 보존 밴드로 펼쳐 개성 유지(완전 회색 수렴 방지). 강도 슬라이더로 조절.
- **가중 역할 배정** — 배경/본문/포인트가 고정되지 않고 조화로운 방향으로 순환.
- **🎲 다시 뽑기** — 색 추출 + 역할 + 무드를 전부 재생성. 같은 입력도 다른 서사.
- **무드 자동 판정 → 형광펜 4타입 + 폰트** — Melancholy→Dialogue, Cozy→Romantic, Vintage→Vintage, Neon→Neon. 같은 핑크라도 무드 따라 형광펜 쓰는 방식이 다름.

파이프라인: `색상 생성 → 무드 판정 → 형광펜 스타일 → 폰트 스타일`.

## 공유
두 탭은 같은 12슬롯·미리보기·**ST 적용(피커 주입)**·라이브러리를 공유. 어느 탭에서 만들든 `ST에 적용`/`💾 저장`으로 나간다.

## 슬롯
네이티브 10슬롯(본문/이탤릭/밑줄/인용/그림자/채팅배경/UI배경/테두리/유저틴트/AI틴트)은 색상 피커에 주입.
커스텀 2슬롯(대사색/형광펜)은 주입된 `<style>`로 적용. 폰트도 같이 주입.

## 설치
`SillyTavern/data/<user>/extensions/theme-gen/`에 폴더째 넣고 ST 재시작. 또는 확장 설치 메뉴에서 GitHub URL.

## 사용
확장 설정(🧩) → "테마 생성기 활성화" + "연결 프로필"(AI용) → "🎨 테마 생성기 열기".
또는 마법봉(확장 메뉴)에서 바로 실행.

## 로드맵
- 성격 태그(temperature/mood/clarity)로 의도적 충돌 조합("비온뒤 노을")
- 라이브러리 export/import
- 무드 판정 정밀화

## ⚠ 설치 후 확인 (버전차로 안 잡힐 수 있는 3곳)
`index.js`의 `[VERIFY]` 주석:
1. import `../` 갯수 (third-party 확장 경로)
2. 연결 프로필 호출(`ConnectionManagerRequestService` → 폴백 `generateQuietPrompt`)
3. `PICKER_MAP`의 색상 피커 id (ST 테마 패널에서 실제 id 확인)
