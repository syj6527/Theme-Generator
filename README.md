# 🌧️ 테마 생성기 (Theme Generator) — `v0.1.0-beta`

SillyTavern 테마를 **색 추천기가 아니라 무드 생성기**로 만드는 확장. 무드를 축으로 분해 → 결정론적 엔진이 12슬롯 팔레트(네이티브 10 + 대사색/형광펜) + 추천 글꼴을 한 세트로 뽑고, ST 테마 색상 피커에 바로 주입한다.

> **베타.** Mood Mode만 들어있다. Object Mode(말차+딸기 같은 소재 기반)는 다음 버전.

## 설치
1. ST → Extensions → **Install extension** → 이 저장소 URL 붙여넣기
   (또는 `public/scripts/extensions/third-party/theme-gen/`에 폴더째 복사)
2. ST 재시작/새로고침

## 쓰는 법
- **설정창**(Extensions 패널): `테마 생성기 활성화` + `연결 프로필`(AI 추출용) 두 개만.
- **열기**: 마법봉 메뉴의 🎨 테마 생성기, 또는 설정창의 "열기" 버튼.
- 패널 안:
  - **AI 연결 토글** — 켜면 텍스트(예: "비 온 뒤 저녁") → 축 자동 추출. 꺼도 전부 동작(오프라인).
  - **무드 축** 슬라이더 / **디저트 프리셋** / **글꼴(무드)** / **12슬롯 직접 편집**
  - **ST에 적용** — 네이티브 10슬롯은 테마 피커에 주입, 대사색·형광펜·글꼴은 커스텀 CSS로 주입
  - **💾 저장** — 라이브러리(확장 설정에 영구 저장). 항목별 적용/불러오기/삭제.

## 동작 원리 (요약)
- **엔진**: 온도→hue, 습도→채도·뿌염, 밝기→라이트/다크, 채도/대비→강조·배경 대비. 결정론적.
- **형광펜**: 보색이 아니라 **시그니처 색의 명도+/채도−** → "같은 장면에서 나온 색".
- **가드**: 본문↔배경 WCAG 대비 보장, 고채도+저광도(피색)만 구제, 틴트 알파 클램프.
- **형식**: 불투명 = `#hex`, 알파(형광펜/틴트/그림자) = `rgba(...)`.

## ⚠ 버전별 확인 지점 (`[VERIFY]`)
ST 버전에 따라 다를 수 있는 부분만 `index.js`에 표시해둠:
1. `import` 경로 (third-party 기준 `../` 갯수)
2. 연결 프로필 호출 (`ConnectionManagerRequestService` → 없으면 `generateQuietPrompt` 폴백)
3. 테마 색상 피커 ID (`PICKER_MAP`) — 현재 ST 기준 매핑
